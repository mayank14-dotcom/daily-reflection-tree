# Daily Reflection Tree

A deterministic reflection system that guides users through structured self-reflection.

## What it does
It asks structured questions across 3 axes:
- Control (Locus of Control)
- Contribution
- Perspective

## Output
At the end, the system gives a reflection summary based on user choices.

## Files
- tree/reflection-tree.tsv → main logic
- tree/tree-diagram.md → flow structure
- write-up.md → design explanation
